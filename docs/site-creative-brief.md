# NeuroHacks Lab — 网站概览与风格板

## 我们是谁
- 使命：用科学且可落地的工具，帮有 ADHD/自闭/感统需求的人把生活和工作变得更可执行、更少羞耻感。
- 受众：成人 ADHD（职场/学生/自由职业者）、家长与教育工作者、懂得自我调节但想要更系统方法的人。
- 定位：Evidence-based（引用研究）、Frustration-aware（理解执行功能困难）、System-first（给流程和模板而非口号）。

## 品牌语气
- 核心感受：平静、可信、无评判。像一位懂神经科学的同事在给你备好工具箱。
- 写作准则：短句 + 直接可执行步骤；每篇都有 “先试 3 分钟版本” 和 “升级版”；避免“全凭意志力”的话术。
- 声音关键词：grounded、compassionate、实验精神、数据旁证、避免过度营销。

## 视觉/体验基调
- 整体气质：Quiet Lab（静谧实验室）+ Soft Tech（柔和科技感），视觉不要过度热闹。
- 色板（建议，用于 Tailwind primary/neutral 调优）：深色 #0F172A（背景/正文），主色 #0EA5E9（强调/交互），暖中性色 #F5EBDD（背景块或卡片底）、辅色 #7C3AED（点缀/标签）。保持高对比度与大量留白。
- 字体：目前使用 `Space Grotesk`，可保持不对称科技感；正文字重用 400/500，标题 600/700；行高 1.6 以上。
- 组件风格：圆角 14–16px、阴影极弱或无阴影；标签保持胶囊形；按钮用填充+细描边两套。
- 图片：避免花哨 stock；多用示意线稿/简笔、手写箭头、流程卡片截图；对产品测评用实拍或可视化评分表。
- 动效：进入时淡入+上移 4–8px；悬停只改轻微阴影/背景，不做夸张缩放。

## 信息架构与内容块
- 顶部导航（建议）：Home / Guides / Tools（工具与模板）/ Reviews（测评）/ Tags / About / Newsletter。
- 首页首屏：1）一句话承诺 + CTA（加入工具箱或订阅）；2）“最近发布”与“精选工具”双列；3）快速入口：按使用场景（上班、课堂、远程、家庭）。
- 文章模板：开头用“场景 + 痛点 + 结果”，每段结束有“立刻试试”微行动；中段用表格/评分卡；结尾有“如果你只记得三件事”。
- 可信度元素：引用 DOI/机构；提供测试流程或测评标准；加上“谁不适合/限制条件”。

## SEO/可见性
- 关键词策略：围绕“quiet fidget toys”“adhd productivity tools”“sensory regulation at work”“adhd study system”等长尾；每篇聚焦一个核心短语 + 2 个情境长尾。
- 结构：标题 <h1> 即关键词短语 + 用户场景；副标题用问题句；列表前 200 字出现关键词；图片添加 alt 描述。
- 内链：每篇至少 3 内链（1 到指南、1 到测评、1 到工具集合）；让标签页成为“微专题”。
